# 微软 CID 获取工具说明
- 本工具用于通过安装 ID（IID）自动获取微软激活确认 ID（CID）。
- 工具采用主备双接口机制，优先请求官方主接口，异常时自动降级至备用接口。
## 运行环境
- 基于 Node.js 环境运行。
- 依赖第三方解析库：xmldom 与 xpath。
## 依赖安装
- 在命令行进入工具脚本所在目录。
- 执行安装命令：npm install xmldom xpath。
## 使用方法
- 命令格式：node getcid.js [安装ID]。
- 示例：node getcid.js 330951690125516940284967699426343120465744714123270282401018483。
- 执行后工具将自动执行并返回最终结果。
## 接口切换逻辑
- 主接口请求超时，自动切换至备用接口。
- 主接口返回 Exceeded（激活次数上限），切换至备用接口。
- 主接口返回 NeverObtained 状态，切换至备用接口。
- 其他情况（如有效 CID、明确错误码），直接返回结果，不切换接口。
## 结果输出说明
- 成功获取 CID：直接输出确认 ID 字符串。
- 主接口错误：输出对应错误码及中文释义。
- 备用接口返回：以 JSON 格式输出完整响应数据
